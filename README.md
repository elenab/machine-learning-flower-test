# Machine Learning with scikit-learn

```
pipenv install scikit-learn seaborn pandas jupyterlab numpy matplotlib pydotplus graphviz
```
Activate the virtual environment:
```
pipenv shell
```
And run:
```
jupyter lab
```
in this environment
